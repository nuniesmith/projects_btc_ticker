## Contributing Images

If you'd like to contribute photos or diagrams:

1. **Photos should be:**
   - Clear and well-lit
   - High resolution (minimum 1920x1080 for photos)
   - Focused on the relevant component/wiring
   - In JPG or PNG format

2. **Diagrams should be:**
   - Vector format (SVG) preferred, or high-res PNG
   - Clearly labeled with component names and pin numbers
   - Use consistent color coding:
     - Red: Power (3.3V)
     - Black: Ground (GND)
     - Yellow: I2C Clock (SCL)
     - Green: I2C Data (SDA)

3. **Naming Convention:**
   - Use lowercase with underscores
   - Descriptive names: `component_description.ext`
   - Examples: `esp32_wiring_top_view.jpg`, `led_resistor_detail.png`
